# Git Features

My dotfiles have a few nice git features you may want to explore.

## Aliases

There are lots of nice `git` aliases.

Some of the most important ones:
```console
$ ga  # git add
$ grm  # git rm
$ gcmsg # git commit -m
$ gp  # git push
$ gl  # git pull
$ gpsup  # git push -u origin $(git branch)
```

For a full list:
```console
$ alias git
```
